# CSS-Parallax-Scrolling-Website

## Project Description
This project is a simple **Parallax Scrolling Website** built using **HTML and CSS**. The website demonstrates a modern **3D parallax scrolling effect** where background and foreground elements move at different speeds to create a depth illusion.

The project highlights different adventure activities like **Biking, Paragliding, and Surfing** while applying parallax animation to enhance the visual experience.

## 🚀 Features
- Smooth **parallax scrolling effect**
- 3D depth illusion using CSS `perspective` and `translateZ`
- Fullscreen hero section
- Adventure themed sections
- Clean and simple UI
- Fully responsive layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- CSS 3D Transforms
- Parallax Scrolling Effect
